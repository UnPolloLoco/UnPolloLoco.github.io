Vim: The Basics

Vim is a text editor that is completely housed within the terminal. It's usually installed by default.

The vim Command

To actually open the text editor, you have to type vim FILE_NAME.
Once the command is run, one of two things will happen:

  1. Vim will open the file and display its contents in the terminal.
  2. If the file doesn't exist, Vim will create an empty file with the given name and open that up instead.

Editing a File

When you use the vim command, it first opens you up in normal mode (more on this later). You cannot directly edit the file yet—not until you enter insert mode. To start typing, hit the i key. Text should appear on the bottom of the terminal window that says -- INSERT --.

Since Vim is a terminal based editor, you can't do much with your mouse other than scroll up and down. Instead, navigate the text using the arrow keys.

Closing a File

Just like any other text editing software, you should save and close a file when you're done with it. To do this in Vim, you have to return to normal mode first—click ESC to do so. Then, type :wq then ENTER to save and quit.

But, you must wonder, what does :wq even mean!? Well, the colon is used to indicate the beginning of a command. Then, "w" means write (save) and "q" means quit.

You have a few options with regards to saving and closing:

  • :wq — Write and quit.
  • :w — Write, but don't quit.
  • :q! — Quit without saving. The "!" lets you bypass warnings.

Other Commands

Remember how I said that commands start with a colon? That's not always true. The following commands—none of which start with a colon—are also used in normal mode.

Do note that these run automatically. No colon means no need to click enter when you're done.

  • x — Delete character
  • dd — Delete whole line
  • yy — Copy (yank) whole line
  • p — Paste
  • u — Undo
  • Ctrl + r — Redo
  • 0 — Move to start of line
  • $ — Move to end of line

Generally speaking, file related commands start with a colon, but text related commands don't.

Final Notes

There's so, so much Vim has to offer. If you want to learn more, a great place to go is Vim's very own built-in tutorial. It can be accessed by typing vimtutor into the terminal. Now, go forth and write!!
